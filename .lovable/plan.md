

## Disable NHL from Daily Parlay Generation

NHL strategies are generated by the `nhl-floor-lock-daily` edge function (scheduled at 12:00 PM ET via pg_cron) and also included in `broadcast-new-strategies`. The simplest approach is to add an early-exit gate at the top of the NHL orchestrator so it skips all generation, and remove NHL strategies from the broadcast filter.

### Changes

**1. `supabase/functions/nhl-floor-lock-daily/index.ts`** — Add early exit
- At the top of the handler (after CORS check), return immediately with a message like `"NHL parlays temporarily disabled"`. This preserves all the code for when you want to re-enable later.

**2. `supabase/functions/broadcast-new-strategies/index.ts`** — Remove NHL strategies from the `.in()` filter
- Remove `'nhl_floor_lock', 'nhl_optimal_combo', 'nhl_ceiling_shot'` and `'cross_sport_optimal', 'elite_cross_sport_6leg'` from the strategy list (lines 31-32) so any stale NHL parlays aren't broadcast.

**3. No UI changes needed** — The `NHLPerformanceCard` and `useDailyParlays` sport tagging can stay as-is; they'll just show no new NHL data.
